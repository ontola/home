Ruby on Rails is een van de meestgebruikte Frameworks om webapplicaties mee te schrijven.
Het is geschreven in de programmeertaal Ruby - een taal die erg makkelijk leesbaar is.

## Waarom Rails gebruiken

- **Veel beschikbare libraries**. Voor Ruby (en Rails) zijn duizenden libraries geschreven, waardoor je voor veel problemen snel een oplossing hebt.
- **Snelle ontwikkeling**. Met Rails kan je in korte tijd grote, relatief complexe applicaties neerzetten.
- **Betrouwbaar**. Rails wordt gebruikt door veel grote softwarebedrijven en heeft zich daarmee flink bewezen.
- **Geschikt voor API's**. Je kunt zowel volledige websites als APIs maken met Rails.

## Combineren met...

Rails is vooral een server applicatie, en heeft (vaak) dus een front-end nodig om de data te tonen.
