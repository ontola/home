Kotlin is een moderne programmeertaal die ontwikkeld is door JetBrains.
Het is compatible met Java, en wordt dan ook vaak gezien als opvolger van Java.
Het draait niet alleen in de Java Virtual Machine (JVM), maar kan ook gecompileerd worden naar Javascript en WebAssembly.
Kotlin is in het bijzonder populair onder Android ontwikkelaars, maar ook in de webwereld wint Kotlin aan popularieit.

## Waarom Kotlin gebruiken

- **Compatible met Java ecosysteem**: Er zijn extreem veel Java packages beschikbaar - dus waarschijnlijk ook voor dat ene ding dat je nodig hebt.
- **Krachtig typing systeem**: Minder bugs en betere onderhoudbaarheid.
- **Compileert naar WASM, Javascript en Typescript**: Zo kan je Kotlin gebruiken taal door de gehele stack - je kunt er zelfs [React](/tech/react) apps mee schrijven!
- **Productiviteit en performance**: Kotlin is een fijne taal om in te werken, waardoor ontwikkeling snel gaat. En dat zonder in te leveren op performance!
- **KTOR als Server framework**: Met KTOR is het schrijven van server applicaties snel gedaan.
